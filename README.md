# Todo CLI

Простое приложение управления задачами из терминала.

## Команды

| Команда | Что делает |
|---------|------------|
| `add "title"` | добавить задачу |
| `list` | показать список |
| `done ID` | отметить выполненной |
| `remove ID` | удалить |

## Запуск

```bash
python main.py add "купить хлеб"
python main.py list
python main.py done 1
```
